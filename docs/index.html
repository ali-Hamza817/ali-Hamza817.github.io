Clinical Note

This tool is for research and academic purposes only. All predictions are generated by models trained on publicly available datasets (SEER, TCGA). The F2-optimized framework penalises False Negatives 4× more than False Positives, aligned with the clinical priority of never missing metastasis. Site-specific scores are relative risk indices, not calibrated absolute probabilities — see the ℹ️ disclaimer in each result panel. Not for clinical diagnosis.
